# ElemonHud
A lightweight desktop tool tailored for teachers to streamline classroom management workflows, including:
- Student homework submission and tracking
- Class participation and behavior point management
- Exam score statistics and average calculation
- Subject-specific auxiliary tools (e.g., grade book, performance tracker)
- Quick access to common classroom management utilities
