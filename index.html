A class homework and points management system built on the Electron framework, designed to support role-separated operations for teachers and students. It streamlines core educational workflows including homework distribution, submission tracking, points statistics, and exam score management.

Key Features

Core Functionalities

  • Teacher password authentication: Differentiates operational permissions between teachers and students to ensure data security
  • Homework Management: Publish or delete homework tasks, assign targeted homework to individual students with corresponding point values
  • Homework Submission: Students automatically earn assigned points upon successful homework submission
  • Data Analytics: Track and manage student points, homework completion status, exam scores, and other academic metrics
  • Student Management: Add new students to the system and update point records in real-time
  • Local Data Storage: Securely store multi-dimensional data (homework, submission records, rewards, redemptions, exams) locally for offline accessibility

UI/UX Features

  • Modern, intuitive UI design with responsive layout support (adapts to different screen sizes)
  • Role-specific interfaces: Dedicated, logically structured dashboards for teachers and students
  • Smooth interactive experience: Visual feedback for user actions (e.g., card hover effects, button animations)

Quick Start

Prerequisites

  • Node.js ≥ 20.0.0 (compatible with Windows 64-bit systems)
  • npm or yarn package manager
  • Electron ≥ 28.0.0

Installation & Run

# Clone the repository
git clone https://github.com/lemonungood/ElemonHud.git
cd elemonhud

# Install dependencies
npm install

# Run in development mode
npm start

# Package for Windows (64-bit)
npm run package

Packaged Build

  • After running npm run package, a standalone Windows executable (.exe) will be generated in the dist/ directory
  • No external dependencies required – the .exe can be run directly on Windows 64-bit systems
  • Zero icon configuration (clean window design with no default app icons)
